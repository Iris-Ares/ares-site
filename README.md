## Development

First, run the development server:

```bash
npm run dev
# or
yarn dev #recommanded
# or
pnpm dev
# or
bun dev 
```

Default boot at [http://localhost:3000](http://localhost:3000)

Modify page : `/app/[lng]/page.tsx`. *([lng] means current page suppot language)*

Modify API:`/app/api`
